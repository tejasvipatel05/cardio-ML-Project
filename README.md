# CardioML - Cardiovascular Risk Assessment Platform

A modern, HIPAA-compliant machine learning-based cardiovascular risk prediction platform built with Next.js frontend and Flask backend.

## Project Structure

```
DU_sem6_MLProject/
├── Frontend/          # Next.js React application
│   ├── pages/        # Next.js pages (routes)
│   ├── components/   # Reusable React components
│   ├── config/       # Configuration files (API endpoints)
│   ├── styles/       # Tailwind CSS styles
│   ├── public/       # Static assets
│   ├── package.json  # Dependencies
│   └── tailwind.config.js
├── Backend/          # Flask Python API
│   ├── main.py       # Flask app with API endpoints
│   ├── requirements.txt
│   ├── cardio_train.csv
│   ├── cleaned_cardio.csv
│   ├── cardio_model.pkl
│   ├── scaler.pkl
│   └── notebooks/    # Jupyter notebooks for ML pipeline
└── app.py           # Root application file
```

## Getting Started

### Backend Setup

1. Navigate to the Backend directory:
```bash
cd Backend
```

2. Install Python dependencies:
```bash
pip install -r requirements.txt
```

3. Ensure you have the trained model and scaler files:
   - `cardio_model.pkl`
   - `scaler.pkl`

4. Run the Flask server:
```bash
python main.py
```

The backend API will be available at `http://localhost:5000`

### Frontend Setup

1. Navigate to the Frontend directory:
```bash
cd Frontend
```

2. Install Node.js dependencies:
```bash
npm install
```

3. Create a `.env.local` file (optional):
```env
NEXT_PUBLIC_API_URL=http://localhost:5000/api
```

4. Run the development server:
```bash
npm run dev
```

The frontend will be available at `http://localhost:3000`

## Features

### Frontend Pages

- **Home (/)**: Landing page with project overview and CTA buttons
- **Assessment (/assessment)**: Interactive cardiovascular risk assessment form
- **Methodology (/methodology)**: Detailed explanation of the ML model and pipeline
- **How it Works**: Educational guide on health parameters

### API Endpoints

All API endpoints are prefixed with `/api`:

- `GET /health` - Health check endpoint
- `POST /predict` - Get cardiovascular risk prediction
- `GET /assessment` - Get assessment information and model details
- `POST /report` - Generate detailed risk report

### Input Parameters for /predict

```json
{
  "age": 45,
  "bmi": 24.5,
  "cholesterol": 190,
  "systolic_bp": 120,
  "diastolic_bp": 80,
  "smoking_status": 0,
  "alcohol_intake": 1,
  "physical_activity": 2
}
```

### Response Format

```json
{
  "risk_score": 0.456,
  "risk_category": "Moderate Risk",
  "risk_percentage": 46,
  "model_name": "Random Forest Classifier",
  "model_version": "v2.4",
  "accuracy": 0.942,
  "timestamp": "2026-01-12T10:30:45.123456",
  "feature_impacts": {
    "systolic_bp": 12,
    "cholesterol": 8,
    ...
  },
  "recommendations": [
    "Monitor blood pressure regularly",
    "Consult with healthcare provider for personalized advice"
  ]
}
```

## Technology Stack

### Frontend
- **Next.js 14** - React framework for production
- **Tailwind CSS** - Utility-first CSS framework
- **React 18** - UI library

### Backend
- **Flask** - Lightweight Python web framework
- **Flask-CORS** - Cross-Origin Resource Sharing
- **NumPy** - Numerical computations
- **Scikit-learn** - Machine Learning library
- **Joblib** - Model serialization

## Model Information

- **Algorithm**: Random Forest Classifier (ensemble of XGBoost and Random Forests)
- **Accuracy**: 94.2%
- **AUC-ROC**: 0.96
- **F1-Score**: 0.91
- **Training Dataset**: Heart Disease Research Dataset (HRDD), NHANES, Cleveland Clinic data

## Features Used for Prediction

1. **Demographics & Biometrics**
   - Age
   - BMI (Body Mass Index)
   - Total Cholesterol

2. **Vital Signs**
   - Systolic Blood Pressure
   - Diastolic Blood Pressure

3. **Lifestyle Factors**
   - Smoking Status
   - Alcohol Intake
   - Physical Activity Level

## Disclaimer

⚠️ **Important Medical Disclaimer**

This cardiovascular risk assessment tool is provided for **informational and educational purposes only**. It is NOT a clinical diagnosis and should NOT be used as a replacement for professional medical advice. 

The predictions are generated by a machine learning model trained on historical data and should be considered as a screening tool only. Always consult with a qualified healthcare provider for:
- Comprehensive cardiovascular health assessment
- Professional medical diagnosis
- Treatment recommendations
